**Generated Code:**
```python
def calculate_median(numbers):
    """
    Calculate the median value from a list of numbers.

    Args:
        numbers (list): A list of numeric values

    Returns:
        float: The median value

    Raises:
        ValueError: If the list is empty or contains non-numeric values
    """
    if not numbers:
        raise ValueError("Cannot calculate median of an empty list")

    # Validate all elements are numeric
    try:
        numeric_list = [float(x) for x in numbers]
    except (ValueError, TypeError):
        raise ValueError("All elements must be numeric")

    # Sort the list
    sorted_numbers = sorted(numeric_list)
    n = len(sorted_numbers)

    # Calculate median
    if n % 2 == 0:
        # Even number of elements
        median = (sorted_numbers[n//2 - 1] + sorted_numbers[n//2]) / 2
    else:
        # Odd number of elements
        median = sorted_numbers[n//2]

    return median

# Example usage
numbers = [3, 1, 4, 1, 5, 9, 2, 6]
print(f"Median: {calculate_median(numbers)}")  # Output: Median: 3.5
```

#### Example 3: SQL Query

**Prompt:**
```
Write a SQL query to find the top 5 customers by total purchase amount in
the last 30 days. Include customer name, email, total purchases, and order
count. Use proper JOINs and date filtering.
```

**Generated Code:**
```sql
SELECT
    c.customer_id,
    c.customer_name,
    c.email,
    COUNT(DISTINCT o.order_id) AS order_count,
    SUM(o.total_amount) AS total_purchases,
    MAX(o.order_date) AS last_order_date
FROM
    customers c
INNER JOIN
    orders o ON c.customer_id = o.customer_id
WHERE
    o.order_date >= DATE_SUB(CURDATE(), INTERVAL 30 DAY)
    AND o.order_status = 'completed'
GROUP BY
    c.customer_id,
    c.customer_name,
    c.email
ORDER BY
    total_purchases DESC
LIMIT 5;
```

### The Anatomy of a Great Prompt

#### 1. Be Specific and Clear

‚ùå **Bad:** "Write about marketing"

#### 2. Provide Context

‚ùå **Bad:** "Explain this concept"

‚úÖ **Good:** "Explain blockchain technology to a 10-year-old child using simple analogies and avoiding technical jargon"

#### 3. Specify Format and Structure

‚ùå **Bad:** "Give me SEO tips"

‚úÖ **Good:** "Create a numbered list of 10 SEO tips for small business websites, with each tip including a brief explanation and an actionable example"

#### 4. Define Tone and Style

‚ùå **Bad:** "Write a product description"

‚úÖ **Good:** "Write a compelling product description for organic coffee beans in an enthusiastic, conversational tone that appeals to health-conscious millennials"


### Advanced Prompt Techniques

#### Chain of Thought Prompting
Encourage step-by-step reasoning:

```
Prompt: "Solve this problem step by step, showing your reasoning at each stage:

A store offers a 20% discount on all items. After the discount, there's an
additional 10% off for members. What is the final price of an item originally
priced at $100?

Let's think through this carefully:"
```

#### Role-Based Prompting
Assign the AI a specific role:

```
Prompt: "You are an experienced financial advisor. A client asks: 'Should I
pay off my student loans or invest in index funds?' Provide a balanced,
professional response considering both options."
```

#### Iterative Refinement
Start broad, then narrow down:

```
First Prompt: "Brainstorm 10 blog post ideas about productivity"
Second Prompt: "Expand on idea #3 with an outline"
Third Prompt: "Write the introduction for this blog post"
```

### Common Prompt Mistakes to Avoid

| Mistake | Why It's Bad | How to Fix |
|---------|--------------|------------|
| **Too vague** | AI makes assumptions | Add specific details |
| **Too long** | Important details get lost | Be concise but complete |
| **No examples** | AI misunderstands format | Include examples |
| **Multiple requests** | Confusing priorities | One clear request |
| **Assuming knowledge** | AI lacks your context | Provide all needed context |
