## Understanding Your Workspace

Your MultinotesAI workspace is designed to keep all your AI-generated content organized and easily accessible.

### Workspace Components

[Screenshot: Workspace overview with labeled sections]

#### 1. Documents Library
- All your AI-generated text content
- Responses, articles, code, and more
- Organized by creation date by default
- Can be moved into folders

#### 2. Folders
- Hierarchical folder structure
- Unlimited nesting depth
- Organize content by project, topic, or client
- Sharable with team members

#### 3. Files
- Your uploaded files (images, audio, video)
- Files used for AI processing
- Organized separately from generated content
- Can be linked to documents

#### 4. Shared Content
- Items shared with you by others
- Items you've shared with others
- View-only or edit permissions
- Separate from your personal content

#### 5. Notebooks
- Collections of related prompts and responses
- Rich text editing capabilities
- Link multiple AI generations
- Perfect for research and projects

### Managing Folders

#### Renaming Folders

1. Right-click on the folder
2. Select "Rename"
3. Enter the new name
4. Press Enter or click "Save"

[Screenshot: Rename folder context menu]

#### Moving Folders

**Method 1: Drag and Drop**
1. Click and hold the folder
2. Drag it to the destination folder
3. Release to drop

**Method 2: Move Dialog**
1. Right-click on the folder
2. Select "Move to..."
3. Choose destination folder from tree
4. Click "Move"

#### Deleting Folders

1. Right-click on the folder
2. Select "Delete"
3. Confirm deletion in the dialog

**Important Notes:**
- Deleted folders are moved to trash (not permanently deleted)
- Content inside folders is also moved to trash
- Recover from trash within 30 days
- After 30 days, deletion is permanent

#### Folder Properties

View and edit folder details:

1. Right-click on folder
2. Select "Properties"
3. View or edit:
   - Name
   - Creation date
   - Number of items
   - Size (if applicable)
   - Sharing status

[Screenshot: Folder properties dialog]

## Managing Documents

Documents are your AI-generated text contentâ€”articles, code, responses, and more.

### Saving AI Generations as Documents

After generating content:

1. Review the AI output
2. Click "Save Document"
3. Fill in document details:
   - **Title:** Give it a descriptive name
   - **Folder:** Select destination folder
   - **Tags:** Add relevant tags (optional)
4. Click "Save"

[Screenshot: Save document dialog]

### Document List Views

#### List View (Default)
- Shows documents as a list
- Displays: Title, Date, Folder, Size
- Sortable columns
- Quick actions (edit, delete, share)

[Screenshot: Document list view]

#### Grid View
- Shows documents as cards
- Displays: Title, Preview, Date
- Better for visual browsing
- Thumbnail previews for images

[Screenshot: Document grid view]

#### Compact View
- Dense list format
- Shows more items per page
- Less detail per item
- Best for quick scanning

### Sorting Documents

Click column headers to sort by:
- **Name** (A-Z or Z-A)
- **Date Created** (Newest or Oldest)
- **Date Modified** (Recently edited first)
- **Size** (Largest or Smallest)
- **Type** (Group by content type)

### Filtering Documents

Use filters to narrow down your documents:

1. Click "Filters" button (top of document list)
2. Select filter criteria:
   - **Date Range:** Last 7 days, Last 30 days, Custom
   - **Content Type:** Text, Code, Image, Audio
   - **Folder:** Specific folder or all
   - **Tags:** Filter by tags
   - **AI Model:** Filter by which model generated it
   - **Shared Status:** Only shared, Only private
3. Apply filters
4. Clear filters when done

[Screenshot: Filter panel with options]

## File Management

Manage your uploaded files (images, audio, video) separately from generated content.

### Uploading Files

#### Method 1: Direct Upload

1. Go to "Files" section
2. Click "Upload" button
3. Select file(s) from your device
4. Choose destination folder
5. Add description (optional)
6. Click "Upload"

[Screenshot: File upload interface]

#### Method 2: Drag and Drop

1. Open the Files section
2. Drag files from your computer
3. Drop them into the browser window
4. Files upload automatically

| Category | Formats | Max Size |
|----------|---------|----------|
| **Images** | JPG, PNG, WebP, HEIC, HEIF, GIF | 250 MB |
| **Audio** | MP3, WAV, AIFF, AAC, OGG, FLAC | 250 MB |
| **Video** | MP4, MOV, AVI, WebM, FLV | 250 MB |
| **Documents** | PDF, DOCX, TXT, MD | 250 MB |

### File Organization

Files follow the same folder structure as documents:

- Create folders specifically for files
- Move files between folders
- Tag files for easy searching
- Link files to related documents

### File Properties

View detailed file information:

1. Right-click on a file
2. Select "Properties"
3. View:
   - File name and type
   - File size
   - Upload date
   - Storage used
   - Used in (which AI generations)
   - Sharing status

[Screenshot: File properties panel]

### Storage Management

Monitor your storage usage:

1. Click on storage indicator (top right)
2. View:
   - Total storage used
   - Storage limit (based on plan)
   - Breakdown by file type
   - Largest files
   - Recent uploads

[Screenshot: Storage usage dashboard]

**Storage Tips:**
- Delete unused files regularly
- Use appropriate image sizes

## Search Functionality

Powerful search helps you find any content instantly.

### Basic Search

1. Click the search bar (top of page)
2. Type your search query
3. Press Enter or click search icon
4. View results

[Screenshot: Search bar with query]

### Search Syntax

#### Simple Search
```
marketing campaign
```
Finds documents containing "marketing" AND "campaign"

#### Exact Phrase Search
```
"email marketing strategy"
```
Finds exact phrase in quotes

#### Exclude Terms
```
marketing -social
```
Finds "marketing" but excludes "social"

#### OR Search
```
marketing OR advertising
```
Finds documents with either term

### Advanced Search

Click "Advanced Search" for more options:

[Screenshot: Advanced search panel]

#### Search Filters

**Content Type:**
- All Content

- Shared items only

**Date Range:**
- Any time

**Folder Location:**
- Search all folders

**AI Model:**
- Any model

## Tags and Categories

Use tags and categories to add metadata for better organization.

### Understanding Categories

Categories are predefined content classifications:
- Content Generation
- Code Generation
- Image Generation
- Audio/Speech
- Analysis

**Categories are:**
- Automatically assigned based on AI feature used
- Used for filtering and organization
- Displayed as badges on documents
- Not editable by users (system-defined)

[Screenshot: Category badges on documents]

### Using Tags

Tags are custom labels you create:

#### Creating Tags

1. Open a document
2. Click "Add Tags"
3. Type tag name
4. Press Enter to add
5. Repeat for multiple tags

**Tag Best Practices:**
- Use lowercase for consistency
- Keep tags short and descriptive
- Use hyphens for multi-word tags (e.g., `social-media`)
- Create a tagging system and stick to it

#### Tag Examples

**By Project:**
- `project-alpha`
- `client-acme`
- `internal-2025`

**By Status:**
- `draft`
- `final`
- `needs-review`
- `published`

**By Topic:**
- `marketing`
- `technical`
- `creative`
- `financial`

**By Priority:**
- `urgent`
- `high-priority`
- `low-priority`

#### Managing Tags

**Tag Manager:**
1. Go to Settings â†’ Tags
2. View all your tags
3. See usage count
4. Rename tags (updates all documents)
5. Delete unused tags
6. Merge duplicate tags

[Screenshot: Tag manager interface]

**Bulk Tagging:**
1. Select multiple documents
2. Click "Add Tags"
3. Type tags to add
4. Apply to all selected

### Filtering by Tags

1. Click "Tags" filter
2. Select one or more tags
3. Documents with ANY selected tag are shown
4. Use "Match all" to show only documents with ALL selected tags

[Screenshot: Tag filtering in action]

### Tag Auto-Suggestions

MultinotesAI suggests tags based on:

## Organization Best Practices

### 1. Establish a Folder Structure Early

Before you have hundreds of documents, decide on an organization system:

**Choose one primary method:**
- Project-based (recommended for freelancers/agencies)
- Content-type based (recommended for content creators)
- Time-based (recommended for regular content production)
- Client-based (recommended for service businesses)

**Stick with your chosen system consistently.**

### 2. Use Descriptive Names

**Good document names:**
- âœ… "Blog Post - SEO Best Practices - Draft 2"
- âœ… "Client Proposal - ABC Corp - 2025-01-15"
- âœ… "Python Function - Data Validation"

**Poor document names:**
- âŒ "Untitled Document 1"
- âŒ "New Document"
- âŒ "asdf"

### 3. Tag Consistently

Create a tagging system and document it:

```
Status Tags: draft, final, needs-review, published
Priority Tags: urgent, high, medium, low
Project Tags: project-[name]
Client Tags: client-[name]
Type Tags: blog, email, social, code, report
```

### 4. Regular Maintenance

Schedule regular cleanup:

**Weekly:**
- Review saved items
- Add tags to untagged content
- Delete unwanted drafts

**Monthly:**
- Archive old projects
- Update folder structure if needed
- Check storage usage
- Clean up trash folder

**Quarterly:**
- Review overall organization
- Update tagging system
- Archive completed projects
- Optimize storage

### 5. Use Search, Don't Over-Organize

Don't spend hours creating perfect folders. Use search and tags to find content quickly.

**Balance:**
- Basic folder structure for big categories
- Tags for detailed organization
- Search for quick access

### 6. Leverage Notebooks for Projects

For any significant project, create a notebook:
- Keeps everything in one place
- Easier to share entire project
- Better for collaboration
- Provides context for content

### 7. Utilize Sharing for Team Organization

**For teams:**
- Create shared folders for team content
- Use consistent naming across team
- Document organization guidelines
- Regular team sync on structure

### 8. Back Up Important Content

Download critical content regularly:
- Use bulk download for projects
- Export as PDF or DOCX
- Keep local backups of essential files
- Export before major changes

### 9. Monitor Storage Usage

Keep an eye on storage:
- Delete large unused files
- Compress images when possible
- Archive old projects
- Upgrade plan if consistently near limit

### 10. Document Your System

Create a simple guide for your organization:
```
My Organization System:

Folders:
- Top level = Clients
- Second level = Project type
- Third level = Content type

Tags:
- Status: draft, final, published
- Priority: urgent, high, normal
- Type: blog, email, social

Naming:
[Type] - [Topic] - [Date] - [Status]
Example: "Blog - SEO Tips - 2025-01-15 - Draft"
```
